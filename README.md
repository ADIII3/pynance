# Prerequisites

To install the required packages run
```
pip install -r requirements.txt
```

# Common tasks

Run the tests.

```
$> doit test
```

Start Jupyter notebook.

```
$> doit notebook
```

## Dash app

To start and show in webbrowser, type

```
$> doit dash_app
```
