# Prerequisites

* [doit](http://pydoit.org) - install `doit`

# Common tasks

Run the tests.

```
$> doit test
```

Start Jupyter notebook.

```
$> doit notebook
```

## Dash app
Make sure you have the following packages installed
* dash
* dash_core_components
* dash_html_component

To start and show in webbrowser, type

```
$> doit dash_app
```
